## Running this project

#### Windows:

```python
pip install virtualenv

virtualenv venv

.\venv\Scripts\activate

pip install -r requeriments.txt

python app.py
```

#### Linux:

```python
pip install virtualenv

virtualenv venv

. .\venv\bin\activate

pip install -r requeriments.txt

python3 app.py
```
